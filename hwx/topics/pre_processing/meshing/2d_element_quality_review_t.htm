Reviewing Element Quality

Use the Element Quality view to review element quality while simultaneously editing your model.

The Element Quality view is automatically enabled when certain Mesh tools are active. To manually enable the Element Quality view, from the view controls (bottom, left of the modeling window), click and select Element Quality.

When the Element Quality view is enabled, elements are colored based on their quality, which is defined by the 2D element criteria settings. A legend displays in the top, left of the modeling window to assist you in evaluating the element quality. A summary of the model's Compound QI and number/percentage of failed elements is also displayed under the legend.

Reviewing 2D Elements by Quality Index

Use the QI (Quality Index) Range legend to investigate the model with overall element quality categorized as Worst, Fail, Warn, Good and Ideal.

  1. In the top, left of the modeling window, click QI Range from the legend.
    The color of elements change according to how well they adhere to the pre-set quality requirements. The quality of the shell model is calculated as a single value based on different categories (worst, fail, warn, good and ideal) defined in the criteria file.
  2. Move the slider to highlight all of the elements at or below a certain quality level.
    For example, setting the slider to Warn highlights all elements in the model whose assessed quality level is Warn or Worst; in other words, elements at the warn, fail, and worst quality levels will be highlighted.
Tip: Change the display of elements that are out of range by right-clicking on the legend and selecting Out of Range from the context menu. This is useful when you want to select elements of interest (for example, only failed elements) by setting the entity filter to Elements and then selecting elements by window or displayed.

Reviewing 2D Elements by Element Criteria

Use the Criteria legend to investigate the model via individual criteria, and view a breakdown of all failed and worst elements based on a set QI range. This is useful when you want to resolve criteria violations, and evaluate the overall quality of a mesh.

  1. In the top, left of the modeling window, click Criteria from the legend.
    The color of the elements change according to the different criteria set in the criteria file. If elements are failing in multiple criteria, then elements will be colored as per their order defined in the criteria file.
  2. Review individual 2D element criteria.
    1. Click on a criteria to review in the legend.
      A single criteria legend displays. Non-editable criteria values are listed from min to max. The colors exhibited in the sliding scale reflect the quality of each element in the modeling window. The elements that are of the best quality always display in blue, whereas, the elements that are of the worst quality always display in red.
    2. Move the slider to view a breakdown of all the failed and worse elements.
      By default, the slider position is based on QI Range legend. For example, if the QI Range legend slider is at the Fail position, then the selected individual criteria, for example Min Size, would have the slider positioned at the failed value for min size as per the criteria file.
    3. Close the single criteria legend by right-clicking on the legend and selecting Close from the context menu, or by clicking < to left of the criteria name.
Tip:
  • Adjust the initial threshold values assigned to each 2D element criteria.
    • In the Criteria legend, click and edit the numbers displayed to the right of each criteria.
    • In the QI Range legend, select a different range.
    • Directly edit the initial threshold values in the Criteria File Editor.
    To restore the default value, delete the number to the right of each criteria and press Enter.
  • Set a legend range for the individual criteria legends by right-clicking on the legend and selecting Set Legend Range from the context menu. The second and second to last values adjust when different ranges are selected, such as good to worst, warn to worst (default), fail to worst, or min to max values. These values are taken from the criteria file, and can be edited. HyperWorks interpolates the values in between these.
  • In the Criteria legend, change the display of elements that have no results by right-clicking on the legend and selecting No Results As from the context menu. In the individual criteria legends, change the display of elements beyond threshold by right-clicking on the legend and selecting Beyond Threshold As from the context menu.

    This is helpful when you want to select elements of interest (for example, only failed elements) by setting the entity filter to Elements and selecting elements by window or displayed.

Configuring the Element Quality View and 2D Element Criteria

Customize the appearance of the Element Quality view, select the criteria that appears in the Criteria legend, and edit the current 2D element criteria.

  • Edit the current 2D element criteria in the Criteria File Editor, which can be accessed by right-clicking on the legend and selecting Edit Criteria from the context menu.
    Note: Any changes that you make in the Element Quality View will impact all of the other Element Quality View settings. The changes that you make to the threshold values in the Element Quality View will not affect the values in the criteria file, but any changes that you make to the criteria file will affect the Element Quality view.
  • Change the color assigned to each element quality category and criteria.
    • In the QI Range and Criteria legends, click on the color next to each element quality category and criteria and select a new color.
    • In the Element Quality View Configuration dialog, accessed by right-clicking on the legend and selecting Configure Quality View from the context menu, click on a criteria's corresponding color and select a new color.
    Respective elements will be highlighted as per the new color scheme.
  • Select 2D element criteria to display in the Criteria legend, and change the order in which they appear using the Element Quality View Configuration dialog, accessed by right-clicking on the legend and selecting Configure Quality View from the context menu.
    • Select the checkboxes for the criteria you wish to display in the Criteria legend.
    • Select a criteria and click / to change the order in which the different criteria appear in the Criteria legend.
    Tip: If you change the order of the element criteria after you have adjusted the threshold values in the Criteria legend, the threshold values will reset to their fail values. It is best to change the order of the element criteria first, and then change the threshold values in the Criteria legend.
