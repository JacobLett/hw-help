HyperWorks 14x.120 Release Notes

What's new and changed in HyperWorks 14x.120.

Platform Support

HyperWorks 2017.1x is now available on Linux.

Graphics

UHD support
Icons and dialogs are now properly scaled on UHD displays.
Tooltip enhancements
When hovering over entities in the modeling window, detailed entity information is displayed in tooltips. You can hide these tooltips by clearing the Tooltips in graphics checkbox in the Preferences dialog. In addition, when hovering over options in dialogs and guide bar menus, option definitions are displayed via tooltips.
Application background color
Changes have been made to the default background color in the application. If you have previously installed HyperWorks 14.x, delete the .altair folder to apply these changes.

File Management

Opening models
Quickly open HyperMesh model files by dragging-and-dropping a .hm file into the modeling window.

View Controls

Reworked View Controls toolbar
The workflow of the View Controls toolbar has been refined, and new geometry and mesh display settings have been added. You can now color both geometry and mesh from the Model Color menu.

Find and Search

Searching for legacy HyperWorks Desktop tools
A new search bar has been added to assist you in finding legacy HyperWorks Desktop tools and panels. You can access the search bar by pressing Ctrl + F. Once enabled, the search bar will open in the top, right of the modeling window.

Translate and Rotate

Rotating the Move tool
You can now double-click the manipulator to enable reposition mode and reposition the Move tool.
Rotate and align the Move tool by clicking-and-dragging the ends of vectors.

Sketch

Sketching in space
When using the Points, Lines, and Circles tools, you can now start your sketch in empty spaces of the modeling window. If you have a model loaded it will use it as a reference when using these tools.
Snapping to global and local inferences
When using the Lines tool, you can now snap to points along the x, y, and z axis. Hovering over snap points along the x, y, and z axes matches the line's color with the default color of the corresponding axes in the lower left corner of the modeling window.
Interpolating points with the Points tool
Interpolate the midpoint of two points or the center of three points by holding Ctrl while creating the points with left-click.
Interpolate multiple points by clicking-and-dragging your mouse to define a start and end point, then specify the number of points to interpolate in the microdialog and press Enter.

Geometry Cleanup

New Cross Extend tool
The Cross Extend tool has been added to the Surface tool group to enable you to extend surfaces to meet each other, or other target surfaces.

Midsurface

New Midsurface tool options
More options from HyperMesh Desktop have been added to the Midsurface tool to assist you in defining how the midsurface is extracted.
Updates to the Edit Plates tool
When you select a midsurface and then switch the Location option to Source geometry, all of the associated source geometry will be automatically selected.

Mesh

Updates to the Surface Mesh tool
The mouse controls used when the Density and Biasing tools are active been modified to improve user interaction.
New Edit Elements tool
A new Edit Elements tool group has been added to enable you to edit 2D surface mesh. The Edit Elements tool group consists of three tools: Split, Collapse, and Move. Split quad elements and edges, combine tria elements, and swap element edges using the Split tool. Collapse element edges using the Collapse tool. Move nodes, drag tria elements, modify holes and washers, and automatically optimize nodes and elements with the Move tool.

Keyboard Shortcuts and Mouse Controls

Configuring mouse control preferences
The default behavior for mouse controls can now be configured in the Preferences dialog. You can now change the default behavior for right mouse drag, middle mouse drag, and left + right mouse drag, or reverse the zoom direction. You can also change the default behavior for appending entities to your selection when in idle mode.
