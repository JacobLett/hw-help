HyperWorks 2017x.1 Release Notes

What's new and changed in HyperWorks 2017x.1.

Workspace

Migration of HyperMesh tools and solver interfaces to HyperWorks
All HyperMesh tools and supported solver interfaces have been added to HyperWorks, with the exception of HyperBeam.
Meshing and model setup tools that are not available in the ribbon can be quickly accessed from the Mesh and Setup drop-down menus below the Model ribbon.
Model ribbon
The Model ribbon contains a variety of meshing and model setup tools, which previously resided in the Geometry ribbon.

File Management

Change solver interfaces
You can now load a different solver interface in HyperWorks, or convert model files from one solver format to another.
Load results
Analysis results can now be loaded in HyperMesh result format (*.res, *.*).
Run command files and scripts
A command file or script can now be run to automate your workflow.
Save screen captures
Screen captures can now be taken of your model and saved to your working directory.

View Controls

Store and recall custom views
The new Views tool enables you to save the current view of your model for future reference, and recall saved views.
In previous version of HyperMesh, views were created, saved, and recalled in the Model browser.
Create and view section cuts
The new Section Cuts tool enables you to create and view 2D section planes of your model.
Switch display between mesh and geometry
You can now switch the display from mesh only, geometry only, or both by pressing G.
Updates to the Element Quality view
The Element Quality view has been updated to add support for the Quality Index Range (QI Range) legend, in addition to the existing Criteria legend. The QI Range legend enables you to investigate the model with overall element quality categorized as Worst, Fail, Warn, Good and Ideal.

Find and Search

Search for HyperWorks tools and panels
A search has been added to assist you in finding tools and panels. You can access the search bar by pressing Ctrl + F, or by clicking in the top, right corner of the modeling window.

Selection

New snap locations
You can now snap to systems (origin), and to line and surface intersections.
Changes to advanced selection
Advanced selection now opens in the Advanced Selection dialog.
When selecting entities, you can now access advanced selection by right-clicking in the modeling window and selecting Select > Advanced from the context menu.

Create Geometry

Toggle circles to between perimeter lines and surfaces
When creating or editing a circle with the Circles tool, you can now toggle the circle between a perimeter line and a surface.
Find and review free edge loops in displayed geometry
A guide bar has been added to the Surface > Patch tool that enables you to find and review free edge loops in all displayed geometry.
Extend discontinuous surfaces
When using the Surface > Extend tool, you can now select and extend any number of surface edges simultaneously, regardless of connectivity, in either tangent or normal directions.
Create solid geometry
The new Primitives tool group enables you to quickly and easily create basic geometry shapes like cubes, cylinders, and spheres.

Geometry Cleanup

Automatic geometry cleanup
The new Auto Cleanup tool enables you to automatically stitch and suppress edges, remove edge fillets, and fix overlapping and duplicate surfaces to prepare your model for meshing.

Midsurface

New plate types
Two new plates types have been added to the Edit Plates tool.
Ignored
A surface that is ignored during midsurfacing.
This plate type is useful when working with model that have section you do not want to midsurface.
Unidentified
A plate that has not been assigned a plate type.
If you do not assign a plate type to unidentified plates prior to midsurfacing, one will be assigned during the midsurfacing process.

Mesh

Auto remesh
Mesh is now automatically refreshed locally after any change is made to edge densities, biasing, or surface mesh styles when using the Surface Mesh tool. This feature is enable by default, but can be turned off in the Options dialog.
Faster copying of edge density and biasing settings
The copying edge density and biasing settings is now faster by using Ctrl + C and Ctrl + V while hovering over edges when using the Surface Mesh tool.
Move multiple nodes in any direction
Besides dragging individual nodes in the mesh plane, you can now select any number of nodes and move them freely using the Edit Elements > Move tool.
Morph by mesh edges and faces
It is now possible to morph by selecting mesh edges and faces directly using the Morph Mesh tool. Edges and faces can be moved freely using the Move tool, or when mapped to selected Targets.
Auto select anchors and morph area
Anchors and morph area are now fully recalculated after each moving entity is either appended or deselected using the Morph Mesh tool. This enhancement allows for faster consecutive morphs with less manual selection, and more intuitive local morphing. This feature is enabled by default, and can be turned off in the Options dialog.
