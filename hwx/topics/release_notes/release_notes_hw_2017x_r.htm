HyperWorks 2017x Release Notes

What's new and changed in HyperWorks 2017x.

File Management

Selecting a user profile
You can now select a user profile when creating a new model from the File menu.

Translate and Rotate

Aligning the Move tool with a selected entities local/global axes
New options have been added to the Move tool to enable you to align the Move tool with a selected entities local or global axes.

Selection

Temporarily hide entities to be isolated
Press Ctrl + H to temporarily hide, but not deselect, selected entities, then press I to isolate your selection in the modeling window.

Create Geometry

Delete surfaces with the Patch tool
You can now delete selected surface when you are using the Patch tool.
New Offset tool
The Offset tool has been added to the Surface tool group to enable you to offset surfaces in a normal direction.

Geometry Cleanup

Imprinting points and surfaces
When using the Imprint tool, you can now imprint points onto lines or surfaces, or surfaces onto surfaces.
Updates to the Stitch tool
You can no longer suppress edges using the Stitch tool.
When using the Stitch tool, you can now repair geometry by replacing edges and points. To repair geometry, click-and-drag an edge to another edge, or click-and-drag a point to another point. You can also select point(s) to move, then click on one of the selected points that you want to retain and combine with the other selected points.
New Suppress Edges tool
The Suppress Edges tool has been added to the Stitch tool group to enable you to suppress or unsuppress edges individually or by break angle.

Midsurface

Clear existing plates prior to extracting midsurfaces
The Clear existing plates checkbox has been added to the Midsurface tool's Options dialog. When this checkbox is enabled, HyperWorks will clear all existing plates prior to extracting midsurfaces.
Updates to Edit Plates tool keyboard shortcuts
When using the Edit Plates tool, you can no longer create plates by clicking the middle mouse button.
New Map Thickness tool
The Map Thickness tool has been added to enable you to automatically apply thickness values to shell elements on midsurfaced geometry (Only supported in the OptiStruct, Nastran, and Abaqus user profiles).

Mesh

New Edit Elements tool
Two new tools have been added to the Edit Elements tool:
  • The Smooth tool enables you to optimize the quality of selected elements.
  • The Create tool enables you to create new elements (1D, 2D tria, 2D quad) and delete existing elements.
New Morph Mesh tool
The Morph Mesh tool has been added to enable you to morph mesh by moving nodes with the Move tool, or by mapping nodes to lines, surfaces, elements, or other nodes.
